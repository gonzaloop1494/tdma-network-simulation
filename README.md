# tdma-network-simulation
Simulación de 10 usuarios en canales Rayleigh (1.8 GHz) y frecuencias Doppler (5, 50, 120 Hz). Compara algoritmos de asignación (SNR vs. utilidad) y distribuciones espaciales. Evalúa retardo, BER y tasa de transmisión con modulación 16QAM y adaptativa (16/64QAM) mediante 10^6 realizaciones de Monte Carlo.
